# 🚨 Red Security Dashboard

A futuristic hacker-style CCTV monitoring dashboard built using HTML, CSS, and JavaScript.

## Features

* 🔐 Animated Login Screen
* 💻 Hacker Terminal Effect
* 📹 6 CCTV Surveillance Panels
* 🚨 Alert Center
* 📊 Live System Logs
* 🕒 Real-Time Clock
* 🌧 Matrix Background
* 📱 Fully Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
